## Personalização rápida

1. Abra `index.html`.
2. Procure por `5554999999999` e substitua pelo número de WhatsApp com DDI e DDD, somente números.
3. Troque `(54) 99999-9999` pelo telefone visível.
4. Atualize os textos, cursos, depoimentos e links.
5. Substitua as imagens da pasta `assets` quando tiver fotos definitivas.
6. Abra `index.html` no navegador para testar.
